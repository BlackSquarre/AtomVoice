## What's New
- **Sherpa Runtime Updates:** Added a dedicated button to check for and install Sherpa-ONNX runtime updates. The app now tracks local versions via a version file and fetches latest releases directly from GitHub.

## Improvements
- **Default Trigger Key:** Changed the default activation key from `Fn` to **Right Option (⌥)** to improve compatibility with standard keyboards and reduce accidental triggers.
- **UI Localization & Terminology:** Standardized the Apple Speech label as "Apple Speech Recognition" and ensured consistent translation across all update dialogs for better clarity.

---

## 新功能
- **Sherpa 运行时更新：** 新增独立更新 Sherpa-ONNX 运行时的功能。应用现在可以通过本地版本文件追踪安装状态，并支持直接从 GitHub 检查并获取最新版本。

## 优化
- **默认触发键变更：** 将默认激活按键从 `Fn` 修改为 **右 Option (⌥)**，以提升在各类标准键盘上的易用性并减少按键冲突。
- **界面本地化与文案优化：** 统一了 Apple 识别引擎的开关标题，并确保所有更新对话框在各语言下的翻译保持一致。
